## 差し替え手順

1. 写真素材を `/public/images/` に配置
2. 各ファイルの `src` プロパティを `/images/ファイル名.jpg` に変更
3. `next.config.ts` の `remotePatterns` は不要になるため削除可能
4. `next/image` の `width` / `height` を実際の画像サイズに合わせて調整
